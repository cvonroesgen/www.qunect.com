Application Note for QuNect ODBC for QuickBase

Quickbase Field Names to Avoid

Please avoid using the following words in your field names or as the name of your field.

  • ADD
  • ALL
  • ALTER
  • AND
  • ANY
  • AS
  • ASC
  • AVG
  • BETWEEN
  • BY
  • CONSTRAINT
  • COUNT
  • CREATE
  • CREATEOQ
  • DELETE
  • DELETEOQ
  • DESC
  • DISTINCT
  • DROP
  • EXISTS
  • FOREIGN
  • FROM
  • GROUP
  • HAVING
  • IN
  • INDEX
  • INNER
  • INSERT
  • INSERTOQ
  • INTO
  • IS
  • JOIN
  • LEFT
  • LIKE
  • KEY
  • MAX
  • MIN
  • NOT
  • NULL
  • ON
  • OR
  • ORDER
  • OUTER
  • PRIMARY
  • REFERENCES
  • SELECT
  • SET
