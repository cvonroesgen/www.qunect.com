QuNect ODBC for QuickBase® is the world's first ODBC driver for Quickbase®.

QuNect ODBC for QuickBase gives you read and write access to your Quickbase data directly from many Windows®, Linux and MacOSX applications like SQL Server, Tableau, Power BI, BarTender, Informatica®, SQL Server Integration Services (SSIS), Pentaho Kettle®, Hummingbird GENIO Suite® Crystal Reports, SPSS®, Business Objects, MS Access , Hyperion Interactive Reporting Studio, SQL Server , Autodesk® Revit Architecture, MS Excel  , MS Word, MS Excel Office 365  .

Programmers can think of QuNect ODBC for QuickBase as the ultimate SDK wrapper for the non-standard Quickbase HTTP API. Access Quickbase easily from PHP, Python, VB.Net, C#, Ruby, VB6, VBA, and Perl, using the industry standard SQL language. Make sure your Quickbase integrations are as maintainable as possible by interfacing with Quickbase through the industry standard ODBC API.
