Application Note for QuNect ODBC for QuickBase

Tableau Desktop

  • Name the file QuNect.tdc and save it to Documents\My Tableau Repository\Datasources.
  • Note: The file must be saved using a .tdc extension, but the name does not matter.
  • Restart Tableau Desktop to apply the change.
  • Create a new connection to QuickBase using QuNect ODBC for QuickBase.
  • Go to and open the Tableau Desktop log file, and look for a record similar to the example below to verify that this customization file was applied to your new connection.

Log File Location:

By default, C:\Users\<user>\Documents\My Tableau Repository\Logs

Example of Record:

Found matching TDC 'C:\\Users\\<user>\\Documents\\My Tableau Repository\\Datasources\\QuNect.tdc' for class='genericodbc', vendor='QuNect for Quickbase', and driver='QuNect ODBC for QuickBase'

Tableau Server

Name the file QuNect.tdc and save it to the following location:

For Windows:

Program Files\Tableau\Tableau Server\<version>\bin

Or

ProgramData\Tableau\Tableau Server\data\tabsvc\vizqlserver\Datasources

To save the file, you must be a Windows Administrator on the server computer.

For Linux:

/var/opt/tableau/tableau_server/data/tabsvc/vizqlserver/Datasources/

Note: The file must be saved using a .tdc extension, but the name does not matter.

Restart Tableau Server to apply the change.

In Tableau Desktop, create a new connection to QuickBase as described in Create a connection.

Go to and open the Tableau Server log file, and look for a record similar to the example below to verify that this customization file was applied to your new connection.

Log File Location

C:\ProgramData\Tableau\Tableau Server\data\tabsvc\vizqlserver\Logs

Example of Record

Found matching TDC 'C:\\ProgramData\\Tableau\\Tableau Server\\data\\tabsvc\\vizqlserver\\Datasources\\QuNect.tdc for class='genericodbc', vendor='QuNect for Quickbase', and driver='QuNect ODBC for QuickBase'
