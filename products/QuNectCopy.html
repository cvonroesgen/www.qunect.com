QuNect Copy

Copy your Quickbase Tables and File Attachments without Limits!

Please install the prerequisite QuNect ODBC for QuickBase first!

Please watch this getting started video 

QuNect Copy allows you to copy any size Quickbase table to another Quickbase table. QuNect Copy will copy file attachments including all revisions, not just the current revision. Control which records and columns are copied by choosing a table report instead of the table as a whole. QuNect Copy is available for a 30 day free trial. After the trial expires a license for QuNect ODBC for QuickBase is required. Licensees of QuNect ODBC for QuickBase can use QuNect Copy for free. QuNect Copy requires MS Windows and .NET Framework 4.5.
