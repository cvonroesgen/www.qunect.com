ODBC for QuickBase

12175 WinHTTPSendRequest A security error occurred

This means your Windows operating system has not been configured for TLS 1.2. You do not need to upgrade QuNect ODBC for QuickBase to achieve compatibility with TLS 1.2.

To configure your Windows operating system for  TLS 1.2, run Windows Update and make sure all updates have been installed. Then click on the blue Download button on this page and run the downloaded installer.
