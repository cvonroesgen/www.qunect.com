Application Note for QuNect ODBC for QuickBase

Listing All Quickbase Applications in SQL Server

If you don't already have a linked server called QUICKBASE, create one using the instructions in the Creating a Linked Server for Quickbase in SQL Server Using the UI application note. To display a list of all Quickbase applications run the following SQL:

sp_catalogs QUICKBASE
