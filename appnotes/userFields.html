Application Note for QuNect ODBC for QuickBase

Quickbase User Field Behaviour

User fields are special in Quickbase. You have to supply an email address or Quickbase screenname. If you insert something like Claude von Roesgen into a Quickbase user field, Quickbase creates a placeholder. Placeholders are indicated by surrounding the text that you insert in double quotes. To learn more about placeholders please read the Quickbase help topic called About Placeholders.
